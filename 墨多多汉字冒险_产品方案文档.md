## 一、产品概述

| 项目 | 内容 |
|-----|------|
| 产品名称 | 墨多多汉字冒险 |
| 产品类型 | 网页版游戏化练字工具 |
| 目标平台 | 手机/平板（主要），电脑（次要） |
| 使用场景 | 每晚9点，15-20分钟 |

## 三、核心问题与解决方案

### 问题拆解

表层问题：写作业拖拉，语文慢
-> 中层问题：写字丑、慢、累，笔顺错误，抵触练字
  -> 根因1：错误笔顺固化（小时候没学好，现在肌肉记忆错误）
  -> 根因2：心理抵触（写字=累+丑+被骂，形成负面关联）
  -> 根因3：缺乏即时反馈（纸笔练习对错延迟，孩子不确定、焦虑）
  -> 根因4：启动门槛高（空白恐惧，不知道从哪里开始）

### 产品解决哪个环节

| 根因 | 产品解决方式 | 不解决的部分 |
|-----|-----------|-----------|
| 错误笔顺固化 | 游戏化高频重复，即时纠错 | 真实握笔肌肉训练（需纸笔配合） |
| 心理抵触 | IP代入、游戏机制、多巴胺奖励 | 深层情绪问题（需家长沟通） |
| 缺乏即时反馈 | 每笔0.5秒判断对错 | 无 |
| 启动门槛高 | 任务拆分、墨多多引导、明确第一步 | 无 |

**关键边界：产品解决"脑中的笔顺记忆"和"愿意练的情绪"，不解决"手部握笔肌肉"。后者必须靠纸笔复刻。**

## 四、产品理念与三阶段递进

> **游戏化是桥梁，不是终点。** 先用电子游戏解决"启动难"和"笔顺记忆"，再逐步过渡到真实纸笔，最终目标是作业本上的流畅正确书写。

### 三阶段递进

```
游戏化兴趣期 -> 双轨融合期 -> 纸笔独立期
（解决启动）    （解决迁移）    （解决日常）
```

| 阶段 | 时间 | 游戏:纸笔 | 核心动作 |
|-----|------|----------|---------|
| 兴趣期 | 第1-2周 | 100:0 | 纯游戏，建立兴趣，熟悉笔顺 |
| 融合期 | 第3-4周 | 70:30->50:50 | 游戏通关后，纸笔复刻3-5遍 |
| 独立期 | 第5-7周 | 30:70->10:90 | 纸笔为主，游戏周末复习 |

## 六、游戏玩法设计

### 单关结构（15分钟）

```
[0:00] 开场动画（墨多多任务发布，30秒）
[0:30] 热身字（2-3个已学字，快速复习，2分钟）
[2:30] 新字教学（2-3个核心字，详细讲解，5分钟）
[7:30] 挑战字（2-3个相似字，辨析对比，3分钟）
[10:30] Boss战（5-10字混合，无提示限时，5分钟）
[15:30] 结算（星级、绿宝石、错误分析，2分钟）
```

### 描红交互细节

| 交互元素 | 定义 |
|---------|------|
| 书写区域 | Canvas自适应，手机竖屏为主 |
| 笔画触发 | touchstart开始，touchend结束一笔 |
| 笔画识别 | 方向匹配（8方向）+ 路径相似度 |
| 撤销机制 | 单笔撤销，整字重写 |
| 错误提示 | 震动+红色闪烁+倒笔怪台词+扶幽数据提示 |
| 容错设计 | 笔画方向偏差30%内算正确 |

### 笔顺判断算法（简化版）

```javascript
// 输入：用户笔画序列 [stroke1, stroke2, ...]
// 每个stroke = 点序列 [(x1,y1), (x2,y2), ...]
// 处理：
// 1. 简化笔画为方向向量（8方向）
// 2. 与预设标准笔画方向序列比对
// 3. 允许顺序错位1位（容错），错位2位+判错
// 输出：正确/错误/顺序错位
```

### 评分系统

| 结果 | 条件 | 星级 | 绿宝石 |
|-----|------|------|--------|
| 完美 | 3遍全对且快 | 3★ | 3/字 |
| 良好 | 3遍全对但慢 | 2★ | 2/字 |
| 及格 | 错1笔顺序 | 1★ | 1/字 |
| 失败 | 错2笔+ | 0★ | 0，强制看提示 |

## 七、关卡与对话脚本

### 第1章：新手村·墨多多训练营

#### 第1关：横画基准（一、二、三、工、王）

**【开场】**
墨多多："欢迎加入冒险队！我是墨多多。汉字大陆被倒笔怪入侵，所有字的骨架都被打乱了。今天，我们从最基础的'横画'开始——一、二、三，这三个字，是整座大陆的基石。记住：横画不只是'一条线'，它是字的'地平线'。地平线歪了，整座城市都会倾斜。"

**【热身字】**
婷婷："'一'字，只有一笔横画。起笔轻顿，行笔平稳，收笔略顿。这是所有横画的母本，像大地，承载一切。"
扶幽："检测：方向正确率100%。无倒笔怪干扰。通过。"

**【新字教学】**
婷婷："'二'字，两层横画。规则：从上到下，逐层搭建。上横短，下横长，中间留空像楼层高度。如果先写下横，上横会压下来，整个字'扁塌'。"
婷婷："'三'字，三层横画。上横最短，中横略长，下横最长。这叫'覆载结构'——上覆下载，像屋顶和地基。三横等距，用眼睛量，像音乐的三拍子：哒-哒-哒。"

**【挑战字】**
婷婷："'工'字，横-竖-横。竖画在正中，像顶梁柱。注意：最后一横比第一横长，像地基比屋顶宽，才稳。"
婷婷："'王'字，横-横-竖-横。三横一竖，竖画贯穿。和'工'的区别：多一横，竖画更长。倒笔怪喜欢让你把'王'写成'工'，少一横偷懒！"
倒笔怪："王和工分不清？哈哈，国王变工匠！"

**【Boss战】**
墨多多："Boss战！倒笔怪放出混合大军！一、二、三、工、王——随机出现，限时15秒，无提示！"
扶幽："实时分析：第1字正确->第2字正确->第3字笔顺错位->修正->通过。"
虎鲨："揍它！别让它跑！"

**【结算】**
墨多多："第1关总结：你掌握了'横画基准'和'层叠结构'。核心规则：从上到下，先上后下。绿宝石：23颗。解锁：墨多多语音'冒险队，出发！'。明日预告：竖画支撑——十、木、本"

#### 第2关：竖画支撑（十、木、本、禾、未）

**【开场】**
墨多多："第2关！横画是地平线，竖画是顶梁柱。没有竖画，字就像帐篷没支柱，风一吹就倒。今天：十、木、本，加上两个'双胞胎'禾、未——倒笔怪最喜欢在双胞胎里捣乱！"

**【新字教学】**
婷婷："'十'字，横竖交叉。核心规则：先横后竖。为什么？横定宽度，竖定高度。先铺地，再立柱。如果先竖后横，柱子没底座，会歪。"
婷婷："'木'字，横-竖-撇-捺。竖画正中，撇捺对称。竖画是脊梁，撇捺是双臂。脊梁正，双臂才能舒展。"
婷婷："'本'字，木+一横。这一横在根部，像树木扎根。注意：最后一横短，在竖画底部，不能出头。"

**【挑战字】**
婷婷："'禾'和'未'，'木'的双胞胎。'禾'：撇在横上，像稻穗下垂。'未'：横在撇捺之间，像树枝分叉。区别：'禾'撇低，'未'横高。倒笔怪让你写混！"
倒笔怪："禾和未分不清？农民哭，树也哭！"

#### 第3关：撇捺平衡（人、八、入、大、天）

**【开场】**
墨多多："第3关！撇捺是字的'双腿'，决定站姿。人、八、入——三个'人形字'，倒笔怪让你站不稳！"

**【新字教学】**
婷婷："'人'字，先撇后捺。撇是发力腿，从右上向左下蹬；捺是平衡腿，从左上向右下舒展。如果先捺，发力腿踢到平衡腿，绊倒。"
婷婷："'八'字，撇捺分开，像双脚开立。和'人'的区别：撇捺不连，各自独立。'入'字，撇捺交叉，像一脚踩进门槛。注意：先撇后捺，交叉点在正中。"

**【挑战字】**
婷婷："'大'字，横+撇捺。横在上，像帽子；撇捺在下，像双腿。'天'字，'大'加一横。注意：两横在上，撇捺舒展。倒笔怪让你把'天'写成'大'，少一横偷懒！"
倒笔怪："天和大分不清？头上少块布！"

#### 第4关：点定位（方、主、玉、太、犬）

**【开场】**
墨多多："第4关！'点'是字的'眼睛'，也是'定位桩'。很多字写不方正，因为点没先定。今天：方、主、玉——三个靠'点'立身的字。"

**【新字教学】**
婷婷："'方'字，点、横、横折钩、撇。为什么先写点？点是定位桩，先打桩，再拉线（横），然后搭框架（横折钩），最后封顶（撇）。如果先写横折钩，桩没打，框架偏移，字歪。"
婷婷："'主'字，点、横、横、竖、横。点是灯塔，三横是阶梯，竖是支柱。注意：三横等距，竖在正中，像塔心。"
婷婷："'玉'字，主+一点。最后一点在右下，像宝石落地。和'主'的区别：多一横，最后一点。"

**【挑战字】**
婷婷："'太'和'犬'，'大'的变形。'太'：大+点。点在撇捺之间，像心口一点。'犬'：大+点。点在右上，像狗耳朵。区别：点位置不同，整个字气质不同。"
倒笔怪："太犬不分？心口长耳朵了！"

#### 第5关：折笔入门+第1章综合（了、子、字、学、孩）

**【开场】**
墨多多："第1章最终战！今天学'折笔'——汉字最难的笔画，因为方向突变。了、子、字，加上挑战：学、孩。然后，Boss战混合前4关全部20字次！"
虎鲨："终于大战了！"

**【新字教学】**
婷婷："'了'字，横撇、竖钩。横撇是一笔！从右上向左下，连贯甩出，像抛鱼竿。然后竖钩，像鱼钩入水。拆开写，鱼竿断，鱼跑。"
婷婷："'子'字，了+一横。横在弯钩中间，像鱼竿上的支架。注意：弯钩要圆润，不能写成直角。"
婷婷："'字'字，宀+子。先写宝盖头：点、点、横钩。然后写子。宝盖是屋顶，子是屋内人。先外后内，像先进门再进屋。"

**【挑战字】**
婷婷："'学'字，觉字头+子。复杂，但结构清晰：先外后内，先上后下。倒笔怪让你跳过宝盖，直接写子！"
婷婷："'孩'字，子+亥。子字旁窄，亥部宽。左右结构，先左后右。子像小孩，亥像大人，小孩站左边。"

**【Boss战·第1章终极】**
墨多多："终极Boss战！倒笔怪放出前4关全部20字次！随机闪现，限时10秒，无提示，无重复！这是真正的冒险——"
扶幽："实时战报：正确率85%...90%...95%..."
墨多多："传奇通关！第1章，完美！你掌握了横画、竖画、撇捺、点、折笔五大基础！获得【木剑皮肤券】+ 绿宝石x30！"
虎鲨："耶！传奇！"
婷婷："基础扎实，第2章会更难。"

### 第2章：迷雾森林·婷婷的词汇迷宫

#### 第6关：左右结构（的、他、你、们、什）

**【开场】**
墨多多："第2章！婷婷在迷雾森林发现新规则：左右结构。左边窄，右边宽，像人侧身站立。今天：的、他、你——三个最高频的左右字！"

**【新字教学】**
婷婷："'的'字，白+勺。左窄右宽，白像脸，勺像耳朵。笔顺：白（撇、竖、横折、横、横），然后勺（撇、横折钩、点）。关键：横折钩是一笔，外框连贯！"
婷婷："'他'字，亻+也。人字旁窄，也部宽。也的笔顺：横折钩、竖、竖弯钩。横折钩一笔，像甩钩子，不能拆！"
婷婷："'你'字，亻+尔。尔部复杂：撇、横钩、竖钩、撇、点。先左后右，人字旁站好，尔部再跟上。"

**【挑战字】**
婷婷："'们'和'什'，人字旁家族。'们'：亻+门。门字框先写竖，再写横折钩。'什'：亻+十。简单，但十在右边要舒展。倒笔怪让你把'们'写成'门'，忘了人字旁！"

#### 第7关：上下结构（是、早、星、旦、晶）

**【新字教学】**
婷婷："'是'字，日+正。先写日：竖、横折、横、横。然后写正：横、竖、横、竖、横。注意：最后两笔撇捺，像双腿撑住整个字。如果先写撇捺，再套日字，像先穿裤子再穿外套。"
婷婷："'早'字，日+十。日在上，十在下。十的竖要短，不能穿过日字，像柱子不能穿透屋顶。"
婷婷："'星'字，日+生。日在上，生在下。生：撇、横、横、竖、横。三横均匀，竖在正中。"

**【挑战字】**

婷婷："'话'字，讠+舌。舌：撇、横、竖、口。先写言字旁，再写舌。言字旁窄，舌宽，像嘴小舌头大。"

**【挑战字】**

## 九、Boss战随机算法

### 核心算法

```javascript
class BossBattle {
  constructor(level, characterPool, playerHistory) {
    this.level = level; // 1-15
    this.pool = characterPool; // 本关可用字池
    this.history = playerHistory; // {字: {correct: n, wrong: n}}
  }

  // 生成Boss战字序列
  generateSequence() {
    const count = this.getQuestionCount(); // 5-10字
    const sequence = [];
    const used = new Set();

    // 1. 必含字：本关新字（必须考）
    const newChars = this.getNewChars();
    for (let char of newChars) {
      if (sequence.length < count) {
        sequence.push({char, type: 'new', weight: 1.5});
        used.add(char);
      }
    }

    // 2. 易错字：根据历史错误率加权
    const errorProne = this.getErrorProneChars(used);
    for (let item of errorProne) {
      if (sequence.length < count) {
        sequence.push({...item, type: 'weak'});
        used.add(item.char);
      }
    }

    // 3. 随机填充：剩余位置从字池随机
    while (sequence.length < count) {
      const char = this.randomFromPool(used);
      sequence.push({char, type: 'random', weight: 1.0});
      used.add(char);
    }

    // 4. 打乱顺序
    return this.shuffle(sequence);
  }

  // 根据关卡确定题目数量
  getQuestionCount() {
    if (this.level <= 5) return 5;      // 第1章
    if (this.level <= 10) return 7;     // 第2章
    if (this.level <= 14) return 8;     // 第3章常规
    return 10;                           // 第15关终极
  }

  // 获取易错字（加权采样）
  getErrorProneChars(excludeSet) {
    const candidates = [];
    for (let [char, stats] of Object.entries(this.history)) {
      if (excludeSet.has(char)) continue;
      const errorRate = stats.wrong / (stats.correct + stats.wrong);
      if (errorRate > 0.3) { // 错误率>30%
        candidates.push({char, weight: errorRate * 2});
      }
    }
    // 按权重排序，取前N
    return candidates.sort((a,b) => b.weight - a.weight).slice(0, 3);
  }

  // 限时计算（毫秒）
  getTimeLimit(char, type) {
    const base = 15000; // 15秒基础
    const difficulty = this.getCharDifficulty(char);
    const levelFactor = 1 - (this.level * 0.03); // 随关卡递减
    return Math.max(8000, base * difficulty * levelFactor);
  }

  // 评分算法
  calculateScore(results) {
    let total = 0;
    let stars = 0;

    for (let r of results) {
      if (r.correct && r.fast) total += 3;      // 对且快
      else if (r.correct) total += 2;           // 对但慢
      else if (r.close) total += 1;             // 接近（错1笔）
      else total += 0;                          // 错
    }

    const max = results.length * 3;
    const ratio = total / max;

    if (ratio >= 0.9) stars = 3;
    else if (ratio >= 0.7) stars = 2;
    else if (ratio >= 0.5) stars = 1;
    else stars = 0;

    return {stars, total, bonus: stars === 3 ? 10 : 0};
  }
}
```

class StrokeEngine {
  constructor(canvas) {
    this.canvas = canvas;
    this.ctx = canvas.getContext('2d');
    this.currentStroke = []; // 当前笔画点序列
    this.strokes = [];       // 已完成笔画
  }

  // 开始一笔
  startStroke(x, y) {
    this.currentStroke = [{x, y}];
  }

  // 移动
  moveStroke(x, y) {
    this.currentStroke.push({x, y});
    this.draw();
  }

  // 结束一笔，识别方向
  endStroke() {
    const direction = this.simplifyDirection(this.currentStroke);
    this.strokes.push(direction);
    return direction; // 如 "right", "down-left"
  }

  // 简化方向：8方向向量
  simplifyDirection(points) {
    const start = points[0];
    const end = points[points.length - 1];
    const dx = end.x - start.x;
    const dy = end.y - start.y;

    const angle = Math.atan2(dy, dx) * 180 / Math.PI;

    if (angle >= -22.5 && angle < 22.5) return 'R';
    if (angle >= 22.5 && angle < 67.5) return 'DR';
    if (angle >= 67.5 && angle < 112.5) return 'D';
    if (angle >= 112.5 && angle < 157.5) return 'DL';
    if (angle >= 157.5 || angle < -157.5) return 'L';
    if (angle >= -157.5 && angle < -112.5) return 'UL';
    if (angle >= -112.5 && angle < -67.5) return 'U';
    if (angle >= -67.5 && angle < -22.5) return 'UR';
  }
}

#### validator.js

```javascript
class Validator {
  standards = {
    "方": {
      strokes: ["点", "横", "横折钩", "撇"],
      directions: ["D", "R", "DR", "DL"]
    }
    // ... 其他字
  };

  validate(character, userStrokes) {
    const standard = this.standards[character];
    const standardDirs = standard.directions;

    if (userStrokes.length !== standardDirs.length) {
      return {
        correct: false,
        stars: 0,
        wrongIndex: -1,
        reason: "笔画数量错误"
      };
    }

    let wrongCount = 0;
    let wrongIndex = -1;

    for (let i = 0; i < standardDirs.length; i++) {
      if (userStrokes[i] !== standardDirs[i]) {
        wrongCount++;
        if (wrongIndex === -1) wrongIndex = i;
      }
    }

    let stars = 0;
    if (wrongCount === 0) stars = 3;
    else if (wrongCount === 1) stars = 2;
    else if (wrongCount === 2) stars = 1;
    else stars = 0;

    return {
      correct: wrongCount === 0,
      stars,
      wrongIndex,
      reason: wrongCount > 0 ? `第${wrongIndex + 1}笔顺序错误` : "正确"
    };
  }
}
```

## 十三、纸笔过渡方案

### 核心原则

> 不是"切换"，而是"融合"，逐步加重纸笔比例。

### 三阶段过渡

| 阶段 | 周次 | 游戏:纸笔 | 核心动作 |
|-----|------|----------|---------|
| 兴趣期 | 1-2 | 100:0 | 纯游戏，建立兴趣，熟悉笔顺 |
| 融合期 | 3-4 | 70:30->50:50 | 游戏通关后，纸笔复刻3-5遍 |
| 独立期 | 5-7 | 30:70->10:90 | 纸笔为主，游戏周末复习 |

### 纸笔辅助工具

| 工具 | 作用 |
|-----|------|
| 握笔器 | 强制正确姿势 |
| 三角杆铅笔 | 自然引导三指定位 |
| 带笔顺透明字卡 | 盖纸上照着写，降低记忆负荷 |
| 米字格/方格本 | 结构有边界，字不歪 |
| 可擦写笔/磁板 | 写错即擦，无痕迹压力 |

### 纸笔游戏化

- 纸笔专用地图：实体贴纸，与游戏地图平行
- 纸笔Boss战：某些字必须纸上写才能击败强化版
- 家长NPC语气："这笔横画方向对了！倒笔怪掉血！"

### 每晚标准流程（20分钟）

```
1. 打开游戏/APP（1分钟）
2. 游戏化学习：新字1个或复习2个（5-8分钟）
3. 纸笔复刻：握笔器+米字格，写3-5遍（5-8分钟）
4. 家长检查：笔顺+姿势，给星级（2分钟）
5. 结算：记录绿宝石，更新地图（2分钟）
6. 预告明天：激发期待（1分钟）
```

## 十五、风险与对策

| 风险 | 表现 | 对策 |
|-----|------|------|
| 手指≠握笔 | 平板会了，纸笔仍错 | 融合期强制纸笔复刻，不跳过 |
| 依赖反馈 | 没有音效不会写了 | 后期关闭提示，引入"默写模式" |
| 游戏疲劳 | 玩腻了，绿宝石失效 | 每月更新奖励机制，加入社交 |
| 姿势更糟 | 低头玩平板，颈椎问题 | 固定支架，视线平视，限时15分钟 |
| 家长耐心耗尽 | 看到纸笔仍慢而急躁 | 后台数据看趋势，不以单日论 |
| 过渡期崩溃 | 退回游戏期 | 降低纸笔比例，稳住再前进 |
| 技术精度不足 | 复杂字误判 | 简化方向匹配，复杂字人工复核 |
| 网易兑换落地 | 需家长手动操作 | 设计兑换券系统，周末集中处理 |
